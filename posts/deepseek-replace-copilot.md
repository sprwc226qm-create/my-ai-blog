## 详细配置步骤

### 1. 安装 Cline

在 VS Code 扩展商店搜索 **Cline** 并安装。安装后侧边栏会出现 Cline 图标。

### 2. 配置 DeepSeek

1. 点击侧边栏 Cline 图标，进入设置面板
2. **API Provider** 选择 `DeepSeek`
3. 填入：
   - **API Key**：你的 DeepSeek API Key
   - **Model**：`deepseek-chat`（编码首选）
4. 点击 **Done** 保存

### 3. 开始使用

在 Cline 对话框里，你可以这样用：

```text
# 代码补全
"帮我补全这个函数，实现二分查找"

# 代码解释
"解释一下这段代码在做什么"

# Bug 修复
"这个函数在处理空数组时会崩溃，帮我修复"

# 重构
"把这个类改成函数式写法"

# 写测试
"为这个模块生成单元测试"
```

## 实际使用技巧

### 1. 善用"上下文"

选中代码后，Cline 会自动把选中的内容作为上下文发给 AI。所以在提问前先**选中相关代码**，回答会精准很多。

### 2. 写好 Prompt

好的提示词是效果的关键。对比：

```
❌ "修复 bug"
✅ "这个函数在处理空列表时在第12行抛出 TypeError，请修复并解释原因"
```

### 3. 多模型搭配

Cline 支持随时切换模型：
- **日常编码**：`deepseek-chat`（够用且便宜）
- **复杂重构**：切到 Claude（推理能力更强）
- **快速问答**：`deepseek-chat`（响应快）

### 4. 不要完全信任 AI

AI 生成的代码一定要 Review，特别是涉及安全、性能的部分。我个人习惯：
1. AI 生成 → 2. 我理解逻辑 → 3. 手动测试 → 4. 提交
