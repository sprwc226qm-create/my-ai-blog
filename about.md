## 这个博客的诞生

这个博客的诞生过程本身就是一次有趣的实验——我在 VS Code 里，借助 **DeepSeek** 和 **Claude** 两款 AI 工具，完成了从项目初始化、内容生成到部署上线的全过程。

整个搭建过程大约花费 1.5 小时，AI 承担了大量重复性工作（配置、排版、代码片段），而我专注于内容方向和个人化调整。

## 技术栈

- **框架**: [VitePress](https://vitepress.dev) — 基于 Vite 和 Vue 的静态站点生成器
- **AI 辅助**: DeepSeek + Claude
- **部署**: Vercel（或其他平台）
- **编辑器**: VS Code
